When I have to use R, Python, SAS, SQL, and Bash at the same time in different projects, to make the transfer easier, I built a list of the commonly used script for reference.

At the beginning, a data set with missing and duplicated values were simulated. This data set was also saved to the feather format, which can be consumed by Python, too.

set.seed(1)
N <- 1000
x1 <- sample(c("A", "B", "C", "D"), N, replace = TRUE)
x2 <- sample(c("A", "B", "C", "D"), N, replace = TRUE)
x3 <- sample(c("A", "B", "C", "D"), N, replace = TRUE)
x4 <- sample(1:10)
x5 <- sample(1:10)
df <- data.frame(x1, x2, x3, x4, x5)

for(i in 1:ncol(df)) { # Some missings
  df[ , i][sample(1:nrow(df), sample(10:50))] <- NA
}

library(feather)
path <- "my_data.feather"
write_feather(df, path)

Missing Value

colSums(is.na(df))

Python

import feather
import pandas as pd
path = 'my_data.feather'
DF = feather.read_dataframe(path)
print(DF.isnull().sum())
## x1    11
## x2    26
## x3    45
## x4    47
## x5    35
## dtype: int64
